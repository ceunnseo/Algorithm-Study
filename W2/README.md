# W2. 스택과 큐

스택과 큐를 직접 구현해보고, stl 함수를 이용하여 스택과 큐의 쓰임새를 익혀봅시다 :)


 <br/>
✅ 기초 : 스택과 큐를 직접 구현해봅시다.

⭐ 필수 : 스터디 전 날까지 고민해보고 풀어봅시다! 스스로 풀지 못하셨다면 다른 사람들의 코드를 참고하여 꼭 공부해봅시다.
